Cat Adoption Manager

About this project

This project is a web app to manage cat adoptions at a pet store. The backend uses Python and FastAPI to communicate with a PostgreSQL database that stores information about the cats.

The frontend uses React to display information in a user-friendly way. Users are able to view all cats in the database, search for specific cats, add or remove cats, and edit cats.
