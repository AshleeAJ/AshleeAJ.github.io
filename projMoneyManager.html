Money Manager application

About this project

This project is a GUI application to help users track their spending.

The first screen is a login screen, which checks the user pin matches to the user pin in the corresponding user number .txt file before allowing login.
The second screen is the main program, allowing users to deposit an amount, then allocate that amount to different types of expenses. The expensives are tracked in a graph below the transaction screen (not pictured).
When the user logs out their activity is written to their user number .txt file so that it may be displayed the next time the user logs in to their account.
