Pizza Order Program

About this project

This project consists of a program to mimic placing an order at a pizza chain, with both a simpler implementation in Python (CLI pictured bottom left), and a more complex implementation in Java (CLI pictured top left).

The Java implementation demonstrates use of classes, methods to get user input and perform functinos, simple JUnit tests, and a text-based menu system to allow the user to make their choices. The github repositories can be found below.
