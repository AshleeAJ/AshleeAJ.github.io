Monster Mayhem Game Manager

About this project

This project is a game manager for a game similar to 'Dungeons and Dragons'. It includes a GUI that allows the user to create a party of custom player characters and pit them against a party of pre-generated monsters. The battle turn sequence is randomly generated; the players must defeat all the monsters to win.
This program attempts to follow coding best-practices through the use of design patterns (such as the MVC pattern) and JUnit testing.
